# TaskManager

A simple C++17 console-based task manager demonstrating OOP and file I/O.

## Build

```bash
mkdir build && cd build

```

## Run

```bash
./TaskManager
```

Tasks are saved to and loaded from `tasks.txt` in the working directory.
